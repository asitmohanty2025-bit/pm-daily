## Product definition

PM Daily is a calendar-led practice system for product managers. It is not a conventional LMS.

```text
Choose schedule
→ verify email
→ choose path
→ create a version-pinned learning plan
→ unlock the Calendar Learning Pass
→ download one complete .ics file
→ open the exact challenge from each calendar event
→ Diagnose
→ Decide
→ Build
→ reflect
→ create and upload evidence
→ track skill and portfolio progress
```

Primary promise:

> Choose when to learn, receive a credible PM path, add the entire syllabus to your calendar, and create visible evidence of skill one challenge at a time.

## Initial commercial model

```text
First 1,000 verified users: explicit free founding claim
After that: ₹199 one-time
Entitlement: lifetime Calendar Learning Pass
```

The free experience includes public discovery, schedule preview, path and syllabus preview, email verification, journey preview, and a sample challenge.

The Calendar Learning Pass unlocks the complete multi-event calendar file, exact daily deep links, plan access, and calendar regeneration.

## Core workstreams

1. Landing page and schedule capture
2. Passwordless authentication and onboarding restoration
3. Path discovery and version-pinned learning-plan creation
4. Journey, Today, Week, Full Path, Artifacts, and Progress
5. Razorpay payment verification and entitlements
6. Complete timezone-aware ICS syllabus generation
7. Diagnose / Decide / Build daily challenge workspace
8. Private uploads, artifacts, and portfolio evidence
9. Versioned curriculum and content operations
10. Strict editable LLM-assisted challenge-generation system
11. Analytics, security, privacy, testing, monitoring, and support

## Launch principle

Calendar value must be visible first. The first paid release is successful only when a learner can configure a schedule, choose a path, pay or claim access, download one valid file containing the complete syllabus, import every event, open the exact daily challenge from each event, and save their work.
